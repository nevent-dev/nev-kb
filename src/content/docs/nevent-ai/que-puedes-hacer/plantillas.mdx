Antes de lanzar un email masivo, siempre conviene revisar la plantilla: que los datos personalizados se muestren bien, que el asunto no esté truncado, que el diseño se vea correcto. Normalmente eso implica abrir el editor, buscar la plantilla, previsualizar y enviarte una prueba.

Con Nevent IA puedes hacer esa revisión desde el chat. El asistente puede previsualizar la plantilla con datos de un contacto real, clonarla para una campaña nueva o enviarte directamente una prueba a tu email.

Sofía, community manager de un club nocturno, tiene cuatro plantillas base que rota cada semana. Antes de usar una, le pide al asistente que se la muestre con los datos de un asistente concreto para comprobar que la personalización funciona.

## Lo que puedes preguntar

Disponible en modo "Solo consultar" y superiores:

- "Lista todas mis plantillas de email"
- "¿Qué plantillas he usado en los últimos 3 meses?"
- "Muéstrame el contenido completo de la plantilla 'Confirmación de compra'"
- "¿Cuántas veces se ha usado la plantilla 'Newsletter mensual'?"
- "Previsualiza la plantilla 'Agenda de la semana' con datos de un asistente real"

## Lo que el asistente puede preparar por ti

<Badge text="Consultar y preparar" variant="tip" /> Disponible en modo STANDARD o superior.

Además de consultar, el asistente puede:

- **Crear una plantilla nueva** — describe el contenido y el asistente genera la estructura base en tu cuenta
- **Actualizar una plantilla** — modifica el asunto, el contenido o las etiquetas de una plantilla existente
- **Clonar una plantilla** — duplica una plantilla para usarla como base de una variante sin modificar el original
- **Renombrar una plantilla** — cambia el nombre sin tocar el contenido
- **Previsualizar con datos reales** — el asistente resuelve las variables de personalización usando un contacto de muestra
- **Enviar una prueba** — manda el email de prueba a hasta 10 destinatarios para validar el diseño en bandeja de entrada real

:::tip[Consejo]
Usa la función de previsualización con datos reales antes de lanzar cualquier campaña con personalización. Así detectas si alguna variable como el nombre o el evento aparece en blanco.
:::

## Así lo usa Sofía antes de cada envío semanal

Sofía tiene una plantilla llamada 'Agenda de la semana' que usa cada miércoles. Antes de crear el borrador de la campaña, quiere asegurarse de que la personalización funciona correctamente.

> "Lista mis plantillas y dime cuándo fue la última vez que usé cada una"

El asistente muestra las cuatro plantillas activas con la fecha del último uso.

> "Previsualiza 'Agenda de la semana' con datos de un asistente real"

El asistente resuelve las variables (nombre, ciudad, próximo evento) con datos de un contacto de la base de datos y muestra cómo quedaría el email en el inbox.

> "Hay una variable que aparece en blanco. Renombra la plantilla como 'Agenda semanal v2' y actualiza el campo de bienvenida para que tenga un texto de reserva si el nombre está vacío"

El asistente actualiza la plantilla.

> "Envíame una prueba a mi email antes de usarla en la campaña"

El asistente envía el email de prueba. Sofía lo revisa en su bandeja, todo correcto.

> "Crea un borrador de campaña usando 'Agenda semanal v2' para el segmento 'Activos últimos 30 días'"

La plantilla revisada queda lista para la campaña del miércoles.

## Lo que no puede hacer todavía

- No puede editar el diseño visual (bloques, colores, imágenes) desde el chat — eso se hace en el editor visual de Nevent
- No puede importar plantillas desde otras plataformas (Mailchimp, Klaviyo)
- La creación de plantillas desde cero es funcional pero produce HTML básico; para diseños más elaborados conviene usar el editor visual
