Configura tu espacio de trabajo en Nevent para que refleje tu marca y se adapte a las necesidades de tu equipo.

## En esta sección

- **Branding**: Logo, colores corporativos, fuentes personalizadas
- **Organización**: Estructura multi-tenant, jerarquías
- **Usuarios y permisos**: Gestión de tu equipo, roles de acceso
- **Configuración legal**: Términos, privacidad, cumplimiento GDPR
- **Redes sociales**: Conecta tus perfiles de Instagram, Facebook, TikTok
- **Facturación**: Métodos de pago, historial de facturación
- **Dominios personalizados**: Configura tu propio dominio con DNS
- **Campos personalizados**: Extiende el modelo de datos con campos adicionales

## Configuración recomendada

Para sacar el máximo provecho de Nevent, asegúrate de:

1. **Subir tu logo y colores** - Para que emails y formularios reflejen tu marca
2. **Configurar campos personalizados** - Captura datos específicos de tu audiencia
3. **Añadir usuarios de tu equipo** - Asigna roles según responsabilidades
4. **Revisar configuración legal** - Asegura cumplimiento con GDPR/LOPD

## Ajustes avanzados

- Configuración de fuentes tipográficas personalizadas
- Dominios propios con certificados SSL
- Estructura organizacional multi-tenant
- Políticas de privacidad y consentimientos
