Use cases are complete workflows, not standalone questions. Each one follows a real person in a specific situation, shows the prompts they used and explains what they got at each step.

If you're looking for a list of questions to copy and paste, go to [Ready-to-use prompts](/en/nevent-ai/ready-to-use-prompts/). If you want to understand the scope of each area, visit [What you can do](/en/nevent-ai/what-you-can-do/).

---

## How to read a use case

Each use case has the same structure:

1. **Who and when** — the person's profile and the starting situation.
2. **Before vs. after** — what they did before Nevent AI and what they do now.
3. **What you need** — access level and estimated time.
4. **Step by step** — the real prompts with what the assistant returns at each one.
5. **Why it works** — the Nevent capabilities behind it.
6. **Variants** — how to adapt the use case to your situation.

## Before you start

To follow any of these use cases you need:

- Nevent AI connected in Claude or ChatGPT. If you haven't set it up yet: [Connect with Claude](/en/nevent-ai/connect-claude/) or [Connect with ChatGPT](/en/nevent-ai/connect-chatgpt/).
- The appropriate access level for each use case (indicated at the start of each one). More info in [Permissions and security](/en/nevent-ai/permissions-and-security/).
